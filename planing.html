a). Objetivo del sitio

Es la creación de un espacio de estudio, donde se integre Objetos virtuales de Aprendizaje OVA. Y objetos Virtuales de Información OVI con el fin de dinamizar las estrategias de aprendizaje en lo referente a Propiedades básicas de CSS, Display: inline, block, inline-block, permitiendo a los profesores interactuar de una manera más dinámica con los estudiantes.


  • Objetivo específico 1

    • Creación de Objetos virtuales de Aprendizaje “OVA” donde el estudiante pueda interactuar con la información, haciendo el aprendizaje más dinámico y atractivo por medio de juegos, trivias foros y chats, teniendo un flujo bilateral de información con el tutor y compañeros

  • Objetivo específico 2

    • Creación de Objetos Virtuales de Información “OVI” con el fin de que cada estudiante cuente con una gran gama de herramientas donde pueda consultar información y profundizar en los conocimientos, se busca diseñar este espacio con el de que contenga videos, libros, presentaciones e información de interés y relevante al curso.

  • Objetivo específico 3

    • Creación de un espacio interactivo entre el estudiante y el profesor donde se pueda compartir el avance y la evaluación obtenida en el desarrollo de los objetivos trasados en el inicio del curso, de igual forma se busca que haya una retroalimentación del aprendizaje hacia el estudiante por parte del profesor con el fin de buscar la mejora continua del estudiante.

PRESENTACIÓN

Tengan un cordial saludo, para el equipo de trabajo de la UNAD es muy grato con tan selecto grupo de estudio, y antes de iniciar a profundizar en :a Propiedades básicas de CSS, Display: inline, block, inline-block, quiero que sepan de antemano que el estándar CSS 2.1 incluye más de 100 propiedades de todo tipo que no permite diseñar el aspecto de las páginas HTML; No obstante, hoy en día los diseños web más actuales muestran recursos gráficos que no se pueden realizar con CSS, como son las sombras, transparencias, esquinas redondeadas y tipografía avanzada entre otros. Por ese motivo, es preciso que los diseñadores web profesionales conozcan las técnicas imprescindibles para crear diseños web avanzados


Teniendo en cuenta lo anterior tendremos los siguientes profundizaremos en las siguientes técnicas imprescindibles:

  • Propiedades shorthand para crear hojas de estilos concisas

  • La propiedad hasLayout de Internet Explorer, imprescindible para solucionar muchos errores de ese navegador

  • Limpiar floats, para trabajar correctamente con los elementos posicionados de forma flotante

  • Cómo crear elementos de la misma altura, imprescindible para el layout o estructura de las páginas

  • Sombras, transparencias y esquinas redondeadas, que no se pueden crear con CSS 2.1

  • Sustitución de texto por imágenes y por Flash, para utilizar cualquier tipografía en el diseño de las páginas.

  • Rollovers y sprites CSS para mejorar el tiempo de respuesta de las páginas

  • Técnicas para trabajar con el texto y la tipografía
