DISEÑOS DE SITIOS WEB

PRESENTACIÓN

Tengan un cordial saludo, para el equipo de trabajo de la UNAD es muy grato con tan selecto grupo de estudio, y antes de iniciar a profundizar en : Propiedades básicas de CSS, Display: inline, block, inline-block, quiero que sepan de antemano que el estándar CSS 2.1 incluye más de 100 propiedades de todo tipo que no permite diseñar el aspecto de las páginas HTML; No obstante, hoy en día los diseños web más actuales muestran recursos gráficos que no se pueden realizar con CSS, como son las sombras, transparencias, esquinas redondeadas y tipografía avanzada entre otros. Por ese motivo, es preciso que los diseñadores web profesionales conozcan las técnicas imprescindibles para crear diseños web avanzados.

OBJETIVO

Aprender los conceptos básicos CSS, para ser aplicados en la creación de páginas interactivas, dándonos la facilidad de aplicar por medio de las propiedades de CSS formatos y diseños a un gran nuero de páginas sin tanto esfuerzo .

MATERIALES

La universidad pondrá a la disposición de los estudiantes una seri de Objetos Virtuales de Información “OVI”, y Objetos virtuales de Aprendizaje “OVA” donde se encontrara la información pertinente sobre las Propiedades básicas de CSS, Display: inline, block, inline-block.
